# Mensaje personalizado

Este es un pequeño script en Python que pide dos datos por consola:
- **Nombre**
- **Género** (h/m)

Con esa información, muestra un mensaje personalizado adaptando la palabra **"Amado"** o **"Amada"** según la respuesta.
